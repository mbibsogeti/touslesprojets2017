Notice : how to use this on Eclipse (in french)

Comment utiliser ce projet Github pour stoquer un nouveau projet Eclipse
- Créer un compte utilisateur Github (gratuit ok)
- Demander en salle son inscription comme "collaborator" de ce projet
- Installer Eclipse Luna+
- Lancer Eclipse
- Passer en perspective Git
- Sur Github, prendre cette adresse en cliquant sur "Clone or Download"
- Dans Eclipse, coller cette adresse dans la zone "Eclipse Repositories"
- - Indiquer login et pass, [next]
- - Choisir la branche Master, [next]
- - Laisser le répertoire par défaut, [finish]
- Dans Eclipse, revenir en perspective Java
- Créer le nouveau projet
- - A la place de la "default location", choisir un sous sous répertoire du
dépot Git local, par exemple C:\Users\mic\git\touslesprojets2017\bob\java_test
(le nom du projet sera donné en salle)
Le projet est désormais dans Eclipse et pret à être envoyé : créer des fichiers,
les ajouter à l'index, et [bt droit]>team>commit...
